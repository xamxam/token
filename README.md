## Xamxam ERC20 Token Smart Contracts
  + Name: ``` Xamxam Token ```
  + Symbol: ``` XAM ```

This token powers the Xamxam knowledge network.

## Motivations
'Xamxam' means 'Knowledge' in sénégalese Wolof, a West African dialect. 'Xam' is the corresponding verb and it means 'To know'.
